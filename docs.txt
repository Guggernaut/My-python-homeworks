1) Структуры данных- это, по сути, и есть структуры, которые могут хранить некоторые данные вместе.\
Другими словами, они используются для хранение свяханых данных.
2) Структуры данных используются практически во всех областях информатики и программирование,\
от операционных систем до интерфейсной разработки и машинного обучения. Структуры данных помогают:
Управлять большими наборами данных и использовать их.
Создавать чёткие иерархические или реляционные связи между точками данных.
Упрощать и ускорять обработку данных.
3) Hash Map- это  тип структуры данных, которая хранит данные в виде пары ключ-значение. Основная цель\
Hash Map- получение данных с помощью ключ.
4) Linked List- это структура данных, который обеспечивает возможность создать двунаправленную очередь их\
каких-либо элементов. Каждый элемент такого списка считается узлом. По факту в узле есть его значение,\
а также две ссылки- на предыдущий и на последующий узлы.
5) Hash Map обычно используется для лёгкого и быстрого извлечения, а Linled List- динамичный доступ и хранение\
